### Week 5

## Task

Build on the understanding from Week 4 by applying monorepo concepts in practice. Migrate the previously published
shared library and micro-frontend applications into a single Nx + pnpm workspace. Set up workspace structure, shared
dependency management, and clear project boundaries. Use Nx features such as dependency graphs and caching to run
targeted builds and tests, and experience how local changes propagate instantly across applications without manual
versioning.
