### Week 4

## Task 

Focus on understanding why monorepos are used, rather than how to fully implement them. Learn the, challenges of managing multiple repositories—such as,
shared library versioning, dependency drift, and, coordination overhead—and how a monorepo approach, addresses these problems. Through hands-on exploration,
with Nx and pnpm, observe how collocating micro-frontends and shared libraries improves developer, workflow, visibility, and build efficiency.
